# Movie & TV App (iOS)

A feature-rich iOS application built with SwiftUI that allows users to browse movies and TV shows, watch trailers,
and save content for offline viewing.

## Features
- Browse trending movies and TV shows
- Search for titles using The Movie Database API
- Watch trailers via YouTube integration
- View detailed information for each title
- Download and save content for offline viewing
- Dynamic and responsive home screen

## Tech Stack
- SwiftUI
- SwiftData (local storage)
- The Movie Database (TMDB) API
- YouTube API
- Xcode
